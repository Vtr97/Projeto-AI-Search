# Projeto AI Search

## Configurar pesquisa

Para configurar uma pesquisa inteligente utilizando o Azure AI Search, siga estes passos gerais:

1. **Criar e configurar os recursos necessários no Azure**: Azure AI Search, Azure AI Services e uma conta de armazenamento.
2. **Fazer upload dos documentos para o armazenamento Azure**: Armazenar os arquivos em um container do Azure Blob.
3. **Indexação e enriquecimento**: Usar o Azure AI Search para indexar os documentos e aplicar habilidades cognitivas.
4. **Consultas**: Realizar buscas no índice criado para extrair informações e testar os resultados.
5. **Revisão do Knowledge Store**: Analisar os dados enriquecidos que foram salvos no Knowledge Store.

## Insights obtidos

Durante o projeto, deu pra perceber bem o quanto o Azure AI Search é poderoso e eficiente para:

1. Extrair palavras-chave
2. Detectar sentimentos
3. Identificar localizações
4. Gerar legendas automaticamente para imagens
5. Organizar dados que estavam sem estrutura

## Aplicações

Tudo isso pode ser muito útil em várias áreas, como:

- Análise de avaliações de clientes
- Atendimento ao cliente
- Pesquisas acadêmicas
- Análise de documentos jurídicos
- Assistentes virtuais

## Conhecimentos obtidos

Depois de concluir o projeto, consegui entender melhor como as diferentes ferramentas do Azure se conectam. Também deu pra ver claramente o quanto uma consulta enriquecida pelo Azure AI Search pode facilitar o trabalho. A automação que ele oferece ajuda muito a aumentar a produtividade na análise de vários tipos de documentos.
